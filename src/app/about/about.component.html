Sobre el proyecto

El proyecto es una aplicación de practica creada para dominar tecnologías reactivas modernas en Angular con RxJS.

Es un CRUD completo sobre datos de clientes (Nombre, apellido, dirección) que demuestra buenas prácticas en estado reactivo, pone énfasis en patrones RxJS, asi como formularios reactivos y consumo de API REST con Firebase Realtime Database, todo con componentes standalone y diseño adaptable.

Funciones

  • Angular v19 (Standalone), usa una arquitectura modular, sin NgModules innecesarios.
  • RxJS, uso de tap, map, pipe, BehaviorSubject, Observable para el manejo de estado y errores.
  • Reactive Forms para validaciones dinámicas y control de estado.
  • Firebase Realtime Data base para el consumo y envió de datos con API REST
  • Responsive y Adaptive design para una buena experiencia en todos los dispositivos.
